Sparebace

sparebace.vercel.app

This project was built in response to a challenge set by Barespace: Transform the way salon bookings are made. Instead of the traditional method, build an AI-powered voice agent that can seamlessly handle bookings with natural voice commands. Sparebace is the resulta reimagined salon booking experience where users interact with a voice-enabled chatbot to schedule appointments in a more natural, hands-free way. The goal was to create a system that feels conversational, responsive, and easier than filling out a form.

Under the hood, Sparebace is built with Next.js 14, Prisma, and PostgreSQL, with Stripe integration for secure payments. Its deployed on Vercel and uses NeonDB for serverless database hosting. The app supports real-time availability checks and a smooth user experience, combining modern web architecture with AI to rethink how bookings should work in a digital-first world.

I accepted an offer from Intercom before completing the interview process at Barespace, but I think it's a nice little project worth showcasing.
